# Product Images

Place your product images in this directory.

## Naming Convention

Images should be named following this pattern:
- `{product-name-lowercase}-{category-lowercase}.jpg`

## Examples:
- `cashews-egg.jpg`
- `badam-egg.jpg`
- `walnut-egg.jpg`
- `kitkat-egg.jpg`
- `oreo-egg.jpg`
- `cashew-eggless.jpg`
- `badam-eggless.jpg`
- `walnut-eggless.jpg`
- `kitkat-eggless.jpg`
- `oreo-eggless.jpg`

## Image Specifications

- Recommended size: 800x600px or 1:1 aspect ratio
- Format: JPG or PNG
- File size: Keep under 500KB for optimal loading

## Fallback

If an image is missing, the website will automatically use a placeholder image from placeholder.com with the product name.
